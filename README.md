Preparação de Dados e Machine Learning

Nesta etapa foram realizadas técnicas de preparação de dados para criação de um modelo preditivo.

Atividades desenvolvidas

* Leitura do arquivo `videos-tratados.snappy.parquet`
* Criação da coluna `Month`
* Indexação da coluna `Keyword`
* Criação do vetor de características com `VectorAssembler`
* Normalização dos dados
* Redução de dimensionalidade com PCA
* Separação dos dados em treino (80%) e teste (20%)
* Treinamento de um modelo de Regressão Linear
* Avaliação do modelo
* Exportação dos dados preparados em formato Parquet
